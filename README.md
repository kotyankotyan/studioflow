# 🎵 StudioFlow DAW

ブラウザだけで動く、初心者向け音楽制作ツールです。

## 使い方

1. 曲ファイル（MP3, WAV, FLACなど）をアップロード
2. 自動でボーカル・ドラム・ベース・その他に分離
3. スライダーで各パートの音量・エフェクトを調整
4. 「書き出し」で完成

## 主な機能

- 🎤 **自動音源分離** — 1曲アップするだけでパート分け
- 🎚️ **かんたんミキサー** — スライダーでバランス調整
- ✨ **プリセット** — ポップ/ロック/EDMなど8種類
- 🔄 **調整前後の比較** — ワンタップで原曲と聴き比べ
- 💾 **自動保存** — ブラウザを閉じても次回復元
- ⚙️ **上級者モード** — 本格的なDAW機能（オートメーション・MIDI変換等）

## 技術スタック

- 純粋なHTML / CSS / JavaScript（依存ゼロ）
- Web Audio API
- IndexedDB（データ永続化）

## ローカル起動

```bash
npx http-server . -p 8080
```
